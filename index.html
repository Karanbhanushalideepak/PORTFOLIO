AI-powered Mock Pitch Voice Agent

Developed a production-grade, conversational AI voice agent that simulates startup investor pitch panels. Built on an asynchronous FastAPI WebSocket streaming pipeline, the system integrates OpenAI Whisper and ElevenLabs to provide real-time, fluid voice interactions. To achieve context-aware questioning and adaptive conversation flows, I engineered stateful, cyclic multi-agent architectures using LangGraph. The entire stack is persisted via MongoDB for performance analytics, containerized with Docker, and deployed on GCP Cloud Run for seamless scalability.

Developed an AI-powered matching engine that couples startup founders with optimal investors using semantic search and dense vector embeddings. The application operates on a lightweight FastAPI backend interface, utilizing MongoDB to manage relational interaction analytics, and is completely containerized with Docker and deployed on GCP Cloud Run for high-throughput scaling.
